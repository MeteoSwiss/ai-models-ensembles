% = 1120 forecasts at 15-day (360 h) lead). Rates and infrastructure facts verified
% against official CSCS / AIL Lugano sources - see notes block at top of

% sources").
%
% Required packages (already in main preamble per experiments_tables.tex):
%   amsmath, booktabs, xcolor[table], array
%
% Reference via \ref{tab:inference-cost}.
%
% Per-member wall-clock formula: (mean job time in min) / (14 init_times) /
% (10 members) for week-helper jobs. Trained-prob baselines (aifsens, atlas,
% fcn3) here use fewer sample jobs in the 14-day window because their
% production runs completed earlier; the per-job mean is still representative.
%
% Total NH per production-baseline run = 8 week-helper jobs x ~3-4 h x 1 node.
% CHF rate: Swiss academia tier 3 (> 10000 NH/yr) at CHF 3.84/NH, with the
% ETH 25% discount applied: CHF 2.88 / node-hour.
% kWh per NH at GH200 node: peak ~5.6 kW per node (4x Grace-Hopper at ~1 kW
% each plus interconnect), conservative average ~4.0 kW under steady inference
% load, x PUE 1.2 -> ~4.8 kWh per node-hour delivered.

  \scriptsize
  $^{\dag}$\,Pure inference seconds per ensemble member per 15-day (360\,h) forecast

  $^{\S}$\,\textbf{CS h} = wall-clock hours for a 50-member case-study
  forecast on a single Alps node (4 GH200 in parallel):
  $50 \times \text{s/mbr} / 4 / 3600$. Atlas's $\sim$8\,h budget at one
  case-study init is the natural cap on how many case studies are feasible

  $^{\star\star}$\,\textbf{aifs\_perturbed} is the AIFS v1 deterministic

  $^{\ast}$\,Single-pass training cost in thousands of GPU-hours, as
  reported by the source paper / model card. Aurora 13K A100-h
  (pretraining 32 $\times$ A100, $\sim$2.5 weeks; fine-tuning not separately
  documented; arXiv:2405.13063). GraphCast 22K TPU v4-hours (32 TPU
  $\times$ 4 weeks; arXiv:2212.12794). SFNO $\sim$4K A100-h: the public
  \texttt{sfno\_73ch\_small} checkpoint's training cost is not separately
  documented, so we use the 256 A100 $\times$ 16\,h figure Mahesh \&
  Pathak report (arXiv:2408.03100) for an SFNO of the same family as an
  order-of-magnitude estimate. AIFS v1 11K A100-h (64 A100 40 GB
  $\times$ 1 week; arXiv:2406.01465). AIFS-ENS 22K H100-h (128 H100 64 GB
  $\times$ 7 days, N320 config; arXiv:2412.15832 Table 3). FCN3 $\sim$80K
  H100-h (Stage-1 pre-training only: 1024 H100 $\times$ 78\,h $=$ 79{,}872;
  the full three-stage training is larger; arXiv:2507.12144).

  $^{\bullet}$\,Atlas training cost is not disclosed (model card
  nvidia/atlas-era5 or arXiv:2601.18111). The benchmark paper reports
  ATLAS-SI at 94\,s per 6\,h step on an A100; our GH200 runs the same
  sampler at 38\,s/step, so the cost is structural to the multi-step
  diffusion sampler, not our setup. NVIDIA also ships a faster
  separately-trained CRPS variant (3.3\,s/step). The 4.2\,B parameter
  count is the 2.4\,B latent predictive transformer (ATLAS-SI) plus the
  shared 1.8\,B decoder, both loaded at inference (arXiv:2601.18111).
